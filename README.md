## Challenge: ChatGPT Generated Videogame

### Objective
Learn how to interact with ChatGPT to make the best out of generative AI in code development.

### Features

#### Game
- Start
- Game Over
- Retry
- Timer
- Music

#### Player
- Spawn
- Movement
- Lives
- Enemies Collision Detection
- Sprite Animation
- Powerups

#### Enemies
- Spawn
- Random Bouncing Movement
- Sprite Animation
